# Sobre o projeto

Simula um sistema simples com agendamentos, ondes os registros são gravados na própria planilha para simular um simples sistema CRUD.
Foi utilizado também um visual mais moderno que o tradicional aplicado pelo VBA.

## Desenvolvimento

A ideia no projeto foi criar classes "globais" que com um fácil ajuste no código podem ser utilizadas para criar novos formulários de maneira simplificada;

- Classe para controle de CRUD do formulário;
- Classe para adicionar máscaras e validações à campos específicos;
- Classe para consulta de "CEP" através da API "ViaCEP";

Este é um projeto "limpo" sem utilização de objetos através de API's externas, somente utilizados os recursos "vanilla" disponibilizado pelo VBE Editor do Excel;
